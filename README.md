## Запуск проекта

Для запуска приложения вам понадобится установленный Docker и Docker Compose.

1. Склонируйте репозиторий:
   git clone https://github.com/CJ-FromOhio/url-shortener.git

2. Создайте файл `.env` в корне проекта и заполните его (используйте `.env.example` как образец).

3. Запустите контейнеры:
   docker-compose up --build
